# C4モデル

## レベル1: System Context (システムコンテキスト)
Kakeibot とユーザー、および外部システムとの全体的な関係性を示します。

```mermaid
C4Context
  title System Context for Kakeibot

  Person(user, "User", "家族（ユーザーA, ユーザーB）。LINEを通じて入出金を入力・ダッシュボードを閲覧する。")
  
  System(kakeibot, "Kakeibot", "LINEメッセージから支出を自動記録し、ダッシュボードで家計を可視化するシステム。")
  
  System_Ext(line, "LINE Platform", "Messaging APIによるWebhook通信と、LIFFによるフロントエンド描画基盤。")
  System_Ext(gemini, "Google Gemini API", "テキストやレシート画像から意図や金額を構造化データとして抽出するAI。")
  System_Ext(sheets, "Google Sheets", "システムの唯一のデータベース（元帳・各種マスタ）。")

  Rel(user, line, "メッセージ送信 / LIFFアプリを開く", "LINE App")
  Rel(line, kakeibot, "Webhookイベント送信 / ユーザー認証", "HTTPS/JSON")
  Rel(kakeibot, gemini, "レシート画像・テキストの解析依頼", "HTTPS/REST")
  Rel(kakeibot, sheets, "トランザクションの追記 / マスタデータの読み込み", "HTTPS/REST (JWT Auth)")
```

## レベル2: Container (コンテナ)
Cloudflare Workers 上で稼働する Kakeibot 内部の論理的なコンテナ構成を示します。

```mermaid
C4Container
  title Container Diagram for Kakeibot

  System_Ext(line, "LINE Platform", "Messaging API & LIFF")
  System_Ext(gemini, "Google Gemini API", "AI Model")
  System_Ext(sheets, "Google Sheets", "Database")

  System_Boundary(kakeibot_boundary, "Kakeibot (Cloudflare Workers / Pages)") {
    Container(dashboard, "Dashboard UI", "React Router v7 / Tailwind", "ユーザーの支出状況や精算タスクを表示するSPA/SSRアプリケーション。")
    Container(api, "Webhook & BFF API", "TypeScript / workerd", "LINEからのWebhook受信、UIへのデータ提供、外部APIとの通信制御を行う中核処理。")
  }

  Rel(line, dashboard, "LIFF経由でホスト", "HTTPS")
  Rel(line, api, "Webhook送信", "HTTPS/JSON")
  Rel(dashboard, api, "データフェッチ / 認証連携", "内部ルーティング")
  Rel(api, gemini, "解析プロンプト送信", "Fetch API")
  Rel(api, sheets, "読み書き (values:append / get)", "Fetch API")
```

## レベル3: Component (コンポーネント)
「Webhook & BFF API」コンテナ内部で具体的に連携する機能モジュール同士のフローを表現しています。

```mermaid
C4Component
  title Component Diagram for Webhook & BFF API Container

  Container_Boundary(api_boundary, "Webhook & BFF API") {
    Component(webhook_endpoint, "Webhook Endpoint", "routes/webhook.ts", "LINEからのリクエストを受信し、署名検証とルーティングを行う起点。")
    Component(ledger_feature, "Ledger Feature", "features/ledger", "トランザクションのパース要求、データ保存、バリデーションなど、家計簿のコアドメインロジックをオーケストレーションする。")
    
    Component(line_client, "Line Client", "infra/line", "LINE APIとの通信（返信処理など）を隠蔽する。")
    Component(receipt_parser, "Receipt Parser", "infra/gemini", "Gemini APIを利用して画像やテキストから構造化データ（金額・カテゴリなど）を抽出する。")
    Component(sheets_storage, "Sheets Storage", "infra/google", "Google Sheetsに対するデータの読み書きとJWT認証を行う。")
  }

  Rel(webhook_endpoint, line_client, "1. 署名検証を依頼")
  Rel(webhook_endpoint, ledger_feature, "2. 解析と記録処理を委譲")
  
  Rel(ledger_feature, receipt_parser, "3. メッセージ/画像の解析を依頼")
  Rel(ledger_feature, sheets_storage, "4. パース済みデータの保存を依頼")
  Rel(ledger_feature, line_client, "5. 処理結果のLINE返信を依頼")
```
